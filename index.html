Awesome SmellAI: Cracking the Code of Everyday Odors – From Capture to Release

Alistair Pernigo (MIT Media Lab)  •  Kaichen Zhou (MIT)  •  Yunge Wen (NYU)  •  Jas Brooks (MIT CSAIL)  •  Paul Pu Liang (MIT)

We present a real-time, end-to-end pipeline for odor capture → recognition → release. Odors are captured via a 4-channel e-nose (NO₂ / C₂H₅OH / VOC / CO), interpreted by ScentRatioNet into mixtures over 12 base scents, and released on a wearable 12-channel Scentrealm neckwear over BLE.

Abstract

Artificial intelligence has transformed HCI in vision and sound, yet smell remains underexplored. We present the first machine learning–driven system that recognizes and generates everyday odors in real time through a portable wearable device. Our pipeline captures odor molecules with an electronic nose, processes them with ScentRatioNet to predict composition across 12 base classes, and reconstructs target smells with a wearable generator. We evaluate with computational benchmarks and human perceptual studies, showing reliable performance on mixtures, and provide design guidelines for future olfactory interaction.

Steps for Inference

Prepare Environment

git clone https://github.com/Alistair0909/SmellAI.git
cd SmellAI

conda create -n smellai python=3.10 -y
conda activate smellai
pip install -r requirements.txt
# minimal:
# pip install numpy pandas matplotlib joblib pyserial torch torchvision torchaudio bleak crcmod

Place Pretrained Files

mkdir -p models
# Copy your files:
# models/mixture_kl_model.pth
# models/channel_scalers_4.joblib

Generate Weights

# Windows example (COM7, 60 s capture, live plot)
python scripts/live_capture_to_weights.py ^
  --port COM7 --baud 115200 --duration 60 ^
  --outdir "data\output" --label "Orange" ^
  --model-path "models\mixture_kl_model.pth" ^
  --scalers-path "models\channel_scalers_4.joblib" ^
  --win-len 600 --hop-len 300 ^
  --topk 6 --temperature 1.0 ^
  --plot-every 2 --allow-pad

Outputs: predictions.csv, device_weights_full/topk.(json|csv), and plots.

BLE Release (Scentrealm Neckwear)

Discover & Smoke Test

# Check services (Nordic UART: 6e4000**)
python scripts/ble_list_services.py --address 08:F9:E0:E4:57:FE

# Quick test (try several channels, adjust handset gain)
python scripts/ble_smoke_test.py --address 08:F9:E0:E4:57:FE ^
  --channel 8 --seconds 8 --handset 230

Send Predicted Weights

# Map top-k weights → seconds (min 1 s per channel)
python scripts/ble_send_weights.py ^
  --address 08:F9:E0:E4:57:FE ^
  --json "data\output\device_weights_topk.json" ^
  --total-seconds 20 --min-per-channel 1 --handset 230

If nothing releases: try a different channel, raise --handset (240–255), confirm cartridges & battery.

Steps for Training

Data: 4-channel time-series CSV @10 Hz; clean baseline/exposure/recovery; labels across 12 base classes + mixtures.

python train.py --config configs/smellai.yaml

# Example config
train_data:
  csv_root:        data/train_csv/
  win_len:         600
  hop_len:         300
  normalize:       per_channel
model:
  arch:            ScentRatioNet
  n_channels:      4
  n_classes:       12
opt:
  lr:              3e-4
  batch_size:      32
  epochs:          100
